|                                            | USE CASE 5                                                                                                                                                |

| **Name**                                   | Move                                                                                                                                                      |
| **Actors**                                 | Player                                                                                                                                                    |
| **Brief Description**                      | The Player can move the pieces according to the game rules.                                                                                               |
| **Preconditions**                          | Use case #4.                                                                                                                                              |
| **Basic flow**                             | The Player drags the pieces to the free spots.                                                                                                            |
